# Input Directory

Place your source files here:

- `audio.mp3` — Audio track (MP3, WAV, FLAC, M4A)
- `cover.jpg` — Cover art image (JPG, PNG, WebP)
- `lyrics.txt` — Plain text lyrics (one line per display)
- `lyrics.srt` — SubRip format (timed subtitles)
- `lyrics.lrc` — LRC format (timed lyrics)

**Rules:**
- Use only files you own or have rights to
- Do not commit audio files to git (they are large)
- This directory is gitignored
- Keep a backup of originals — scripts do not modify source files
